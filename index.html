Open Children's Media Index

A transparent, empirically grounded database of sensory-load profiles for children's television. Every metric is measurable, every score shows its component parts.

This index applies the Children's Media Analysis Toolkit (CMAT) to publicly available programming. Analysis is automated and reproducible. This project does not issue verdicts on appropriateness — it presents labeled measurements that researchers and caregivers can interpret in context. All findings are correlational.
